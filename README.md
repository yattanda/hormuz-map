## プロジェクト概要

**ホルムズ海峡**をめぐる複雑な国際情勢を、**地図**と**図解**でわかりやすく解説する地政学メディアです。  
世界の原油輸送の約20%が通過するこの海峡で「今」何が起きているのか、リアルタイムで読み解きます。

### このサイトで得られること

- **地図で理解するホルムズ海峡** - 位置関係・距離感・戦略的意味を直感的に把握
- **毎日更新の最新情勢** - イランと米国の対立をリアルタイムで追跡
- **タンカー航路・シーレーン解説** - 海上交通路の脆弱性と地政学リスクを可視化
- **原油価格と日本経済** - ホルムズ危機が日本に与える影響をデータで分析

## 情報の信頼性について

### 主な情報源
- 各国政府・国際機関の公式発表
- 信頼性の高い報道機関（Reuters, BBC, AP等）
- シンクタンク・研究機関のレポート

### 編集方針
- **政治的中立性**: 特定国家・政党の立場に偏らない客観的分析
- **複数視点**: 対立する当事者それぞれの論理を紹介
- **事実重視**: 推測と事実を明確に区別
- **訂正対応**: 誤りが判明した場合は速やかに訂正・謝罪
