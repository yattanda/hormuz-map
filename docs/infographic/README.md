# ホルムズ海峡危機インフォグラフィック HTML文字レイヤー叩き台 v0.4

## 目的
生成画像に日本語を焼き込まず、背景画像の上にHTML/CSSで文字・見出し・吹き出し・注釈を重ねるための叩き台です。

## v0.4の主な変更
- 全体を「6枚構成」に整理
- 4A：直接当事者、4B：静かな大国・隣接リスクに分割
- 4Bにロシア、パキスタン、中国、インド、トルコを配置
- パキスタンは「仲介役であり、火元に近い隣国」として表現
- LPG/NGLを2枚目・4Bインド欄・5枚目に反映
- PC版の文字量を20〜30%圧縮
- スマホ用のカード分割を初期実装

## ファイル構成

```text
hormuz-infographic-layer-draft-v04/

  assets/
    slide-01-bg.png
```

## 編集の基本
- 文言：`index.html` 内の該当テキストを直接修正
- 位置：CSS内の `#s01 .t1 { left:...; top:...; width:...; }` の％を調整
- 背景画像：`assets/slide-xx-bg.png` を差し替え
- スマホ：現状は横スクロール表示＋下部カード分割。次段階で本格的にスマホ専用構成へ移行可能

## 注意
- 背景画像は1ページ1ファイルのまま管理してください。
- 文字は画像化しない前提です。
- セリフは実際の発言引用ではなく、各国の立場を短く要約した演出です。
- 数値は表示用の概数です。公開前にEIA/IEA/Kpler等の出典表記を最終確認してください。
