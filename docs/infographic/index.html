2-1 原油・LNG

ホルムズは原油・石油製品とLNGの大動脈です。特にLNGは地理的に代替航路が乏しい。

2-2 LPG/NGL

LPGは静かな第3リスクです。家庭燃料・石化原料・インドの家計負担に効きます。

2-3 市場の反応

市場が恐れるのは「止まること」だけでなく「読めなくなること」。保険料・運賃・在庫行動が先に動きます。

2-4 伝播

海峡緊張 → 保険・運賃上昇 → 原油・LNG・LPG高 → 電力・化学・物流 → インフレ圧迫。

3-1 軍事リスク

機雷、無人機、ミサイル、拿捕、AIS混乱が重なると、全面封鎖でなくても航行判断が厳しくなります。

3-2 本当の厄介さ

低コストの嫌がらせが、掃海・護衛・保険・待機という高コストの混乱を生みます。

3-3 示唆

船会社は「危ないかもしれない」と判断した時点で回避します。封鎖宣言の有無だけを見てはいけません。

4A-1 直接当事者

イラン、米国、イスラエル、湾岸諸国は、それぞれ狙いが違います。共通利益はあっても優先順位は一致しません。

4A-2 読み筋

本当の焦点は「誰が勝つか」より、各国が損失を避けつつ相手に圧力をかける限定戦の管理です。

4A-3 リスク

誰も全面封鎖を望みにくい一方、誤算や限定衝突の連鎖が市場危機を拡大させます。

4B-1 静かな大国

ロシア、中国、インド、パキスタン、トルコは直接撃つ国ではなくても、危機の出口を複雑にします。

4B-2 パキスタン

パキスタンは仲介役になれる一方、地理的にも政治的にも火元から安全な距離にはいません。

4B-3 インドとLPG

インドは原油だけでなく、LPG高が家計・補助金・政治に響きます。ここが見落とされがちな急所です。

4B-4 トルコ

トルコは今回目立ちにくいが、NATO・中東・黒海をまたぐ複線外交を持つため、希少性のある視点です。

5-1 市場プレイヤー

危機を長引かせるのはミサイルだけではありません。買い手、保険、海運、市場インフラが危機を増幅します。

5-2 見えない封鎖

航路が物理的に開いていても、保険がつかない、船員安全を確保できない、荷主が嫌がるなら船は動きにくい。

5-3 LPGの位置づけ

原油はインフレ、LNGは電力、LPGは家計と石化に効きます。報道量より影響経路を見るべきです。
