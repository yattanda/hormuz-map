## [S10] news_data.json 更新メモ

**更新方法：** `docs/data/news_data.json` を最新版（別ファイル・push済み）に置き換え済み。

- `latest` 4件：本日5/4の新情報4件に差し替え済み
- 旧 `latest`（5/3 08:59版）4件：`archive` 先頭バッチ「2026/05/03 08:59 更新分（アーカイブ）」として移動済み
- `osint`：Al Jazeera「Project Freedom」最新記事（isLatest: true）に更新済み
- `updated`：「2026年5月4日 11:13 日本時間JST」に更新済み
- `staleNotice`：空文字（新情報あり）

**⚠️ 注意：** JSONファイルはauto_push.pyで先にpush済みのため、Claude Codeはcommit時にconflictが発生した場合は `git checkout origin/main -- docs/data/news_data.json` で解決すること。
